# 回响项目 - 模型调用方案

## 模型配置

| 组件 | 阿里云百炼模型 | 用途 |
|-----|--------------|------|
| ASR | `qwen3-asr-flash` | 语音转文字 + **语音情感识别** |
| 分析 | `qwen-plus` | 单条情绪/标签提取 + **输入类型判断** |
| 日记生成 | `qwen-plus` | 每日日记生成 |
| 深度智能 | `qwen-max` | 月度/年度复盘、认知洞察 |

## API 返回结构

### ASR 调用（qwen3-asr-flash）

**返回格式**：
```json
{
    "text": "转写的文本内容",
    "asr_emotion": "neutral"  // ASR 语音情感
}
```

**ASR 情感类型（7种）**：
| 情感 | 说明 |
|-----|------|
| happy | 愉快 |
| sad | 悲伤 |
| angry | 愤怒 |
| fearful | 恐惧 |
| surprised | 惊讶 |
| disgusted | 厌恶 |
| neutral | 平静 |

### 文本情绪分析（qwen-plus）

**返回格式**：
```json
{
    "emotion": "positive/neutral/negative",
    "emotion_score": 0.8,
    "mixed_emotions": {"满足": 6, "疲惫": 7},
    "primary_emotion": "疲惫",
    "input_type": "情绪表达",  // 新增：输入类型
    "triggers": ["项目上线"],
    "unspoken_need": "希望付出被看见",
    "energy_level": 4,
    "brief_summary": "一句话概括",
    "keywords": ["工作", "朋友"],
    "topics": ["社交", "休闲"]
}
```

**文本情绪类型（17种）**：
| 大类 | 具体情绪 |
|-----|---------|
| positive | 开心、兴奋、满足、期待、感激 |
| neutral | 平静、困惑、麻木 |
| negative | 疲惫、焦虑、低落、愤怒、失望、恐惧、孤独、压力、烦躁 |

**输入类型（input_type）**：
| 类型 | 说明 | 处理方式 |
|-----|------|---------|
| 情绪表达 | 用户在表达情绪感受 | 深度情绪分析 |
| 回忆过去 | 回忆以前的经历 | 分析回忆中的情绪 |
| 观点分享 | 分享经验、想法、知识 | neutral + 平静 |
| 事实陈述 | 只说发生了什么 | neutral + 平静 |
| 计划目标 | 说接下来要做的事 | neutral + 平静 |

## 数据库字段

**voice_records 表新增字段**：
| 字段 | 类型 | 说明 |
|-----|------|------|
| asr_emotion | VARCHAR(20) | ASR 语音情感 |
| input_type | VARCHAR(20) | 输入类型 |
