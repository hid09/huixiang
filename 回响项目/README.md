# 回响（Echo）项目文档

> **Slogan**：回响，陪你听见自己

---

## 项目简介

「回响」是一款基于语音输入 + AI 智能分析的个人成长记录工具，通过四维度（情绪/关注点/行为/认知）追踪，让用户看见自己的变化，辅助自我成长。

### 核心价值

- **低门槛记录**：语音输入，随时随地，无压力
- **情绪适配**：AI 根据当日情绪动态调整日记语气
- **四维度追踪**：情绪、关注点、行为、认知全面分析
- **认知变化洞察**：看见"过去的我"与"现在的我"的差异
- **温和成长建议**：具体、可执行、不强迫

## 核心功能

### 1. 碎片语音输入
- 随时发送语音，自动转文字
- 多通道：微信小程序/独立 App/网页
- AI 自动打标签（四维度）
- 云端存储

### 2. 每日智能日记
- 每晚 22:30 自动生成
- 情绪识别 + 语气适配
- 四维度分析
- 认知变化检测

### 3. 周期性复盘
- 周回顾（简洁版）
- 月度复盘（深度版 + 建议）
- 季度总结（趋势分析）
- 年度回顾（仪式感）

### 4. 认知变化洞察
- 「过去的我说」功能
- 观点变化时间线
- 成长轨迹可视化

## 核心设计理念

1. **记录是成长的镜子**：不只是存储数据，而是让用户通过记录看见自己
2. **降低输入门槛**：不要求"写得好"，只要有输入就行
3. **有温度的对话**：不是冷冰冰的总结，而是有温度的对话
