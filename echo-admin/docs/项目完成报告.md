## 📋 部署说明

### 环境要求
- Python 3.10+
- Node.js 18+
- MySQL 8.0+

### 后端部署
```bash
cd backend
pip install -r requirements.txt
python init_admin.py  # 创建默认管理员
uvicorn app.main:app --host 0.0.0.0 --port 8100
```

### 前端部署
```bash
cd frontend
npm install
npm run build
# 将 dist/ 目录部署到 Nginx
```

### 默认账号
- 用户名：**admin**
- 密码：**admin123**
