# 回响后台管理系统 - 架构设计文档

## 1. 项目概述

### 1.1 项目信息
- **项目名称**：回响后台管理系统（echo-admin）
- **技术栈**：
  - 前端：React 18 + Ant Design Pro 6.x + TypeScript
  - 后端：FastAPI + SQLAlchemy 2.x
  - 数据库：MySQL（只读账号访问回响项目数据）
- **端口**：8100（后端）、前端由 Nginx 托管

### 1.2 核心功能模块
| 模块 | 功能 | 权限 |
|-----|------|------|
| 登录模块 | 账号密码登录 + 退出登录 | 无 |
| 数据看板 | 核心指标、AI健康度、7天趋势图 | 超管/普通 |
| 用户管理 | 用户列表、用户详情（录音/日记记录） | 超管/普通 |
| 系统设置 | 管理员账号管理 | 超管 |

- **方案**：JWT (JSON Web Token)
- **Header**：`Authorization: Bearer <token>`
- **过期时间**：24 小时

## 5. 数据库设计

### 5.1 自有表：admin_users
```sql
CREATE TABLE admin_users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    role ENUM('super', 'viewer') DEFAULT 'viewer',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_username (username)
);
```

### 5.2 只读表（来自回响项目）
- `users`: 用户表
- `records`: 录音记录表
- `diaries`: 日记表

## 7. 部署配置

### 7.1 目录结构
```
/opt/echo-admin/
├── backend/
│   ├── app/
│   ├── requirements.txt
│   └── main.py
├── frontend/
│   ├── dist/
│   └── nginx.conf
└── supervisor.conf
```

### 7.2 Nginx 配置
```nginx
server {
    listen 80;
    server_name admin.echo-app.com;

    location /api {
        proxy_pass http://127.0.0.1:8100;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location / {
        root /opt/echo-admin/frontend/dist;
        try_files $uri /index.html;
    }
}
```

### 7.3 Supervisor 配置
```ini
[program:echo-admin]
command=/opt/echo-admin/backend/venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8100
directory=/opt/echo-admin/backend
user=www-data
autostart=true
autorestart=true
```

## 8. 技术决策

| 决策点 | 选择方案 | 理由 |
|-------|---------|------|
| 认证方式 | JWT | 无状态、跨域友好、适合前后端分离 |
| 前端框架 | Ant Design Pro | 企业级组件库、开箱即用 |
| 图表库 | Apache ECharts | 功能强大、中文友好 |
| 密码哈希 | bcrypt | 安全、防彩虹表攻击 |
| 数据库迁移 | Alembic | SQLAlchemy 官方迁移工具 |
